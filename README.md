# vending-machine
Python으로 만든 자판기 프로그램 


## 비즈니스 로직
* 음료의 종류는 세가지(코코팜, 코카콜라, 토마토주스)이다.
* 관리자의 경우 음료의 수량을 늘리거나 줄일 수 있다.
* 고객의 경우 음료수의 수량이 0보다 클 경우 에만 음료수를 주문할 수 있다.
